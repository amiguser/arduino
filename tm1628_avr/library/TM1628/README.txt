TM1628 library
--------------
Original: Vasyl Yudin
Current: Amiguser
Email: amiguser(at)gmail(dot)com
URL: 

A library for interacting an arduino and AVR mc with a TM1628.

Includes:
- Support for the TM1628;
- Reading simultaneous button presses;

USAGE NOTES
-----------

Just put the files on a TM1628 directory under "arduino/libraries" on your arduino IDE instalation
